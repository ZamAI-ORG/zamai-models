# 📊 Analysis Scripts

This directory contains scripts for analyzing datasets and models.

## Scripts

### `analyze_zamai_dataset.py`

Comprehensive analysis of your ZamAI Pashto dataset.

- Dataset statistics
- Content analysis
- Language detection
- Instruction type classification

**Usage:**

```bash
python analyze_zamai_dataset.py
```

### `explore_dataset.py`

Interactive dataset exploration.

- Sample data viewing
- Structure analysis
- Format validation

### `explore_zamai_dataset.py`

Specific exploration for ZamAI dataset format.

- Instruction-following format analysis
- Pashto content ratio
- Quality metrics

### `debug_models.py`

Model debugging and diagnostic tools.

- Error analysis
- Performance profiling
- Resource usage monitoring

## Output

Analysis results are saved to `../../data/processed/` with detailed reports and visualizations.
